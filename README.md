# nicovideo-player-expander
[ニコニコ動画](https://www.nicovideo.jp/video_top)のプレイヤーを2種類の方法（「シアターモード」または「ブラウザ内最大化」）で拡大します。
プレイヤー右下のアイコンでこれらの機能を切り替えられます。
それぞれtキー、bキーでも（ブラウザ内最大化解除はescキーでも）切り替えられます。
