# nicovideo-player-expander
[ニコニコ動画](https://www.nicovideo.jp/video_top)のサイドバーを下に移動し、ウィンドウサイズに合わせてプレイヤーを拡大するUserScriptです。
プレイヤー右下の全画面表示アイコンの左隣のアイコンでこの機能のON/OFFを切り替えられます。
